# 達芬奇 DaVinci Resolve 21 AI 自動對拍電影感剪輯系統
## 📘 專案藍圖與開發者技術手冊

本專案是一套高效能的 **AI 智慧音訊對拍與電影感視訊自動剪接系統**。該系統直接與 **DaVinci Resolve 21** 的 Python API 進行物理互動，自動將原始相機素材依照背景音樂的鼓點暫態進行對拍，並以極致精準的節奏自動剪接至時間軸上，完美解決了跨影格率抽影格相位差、訊號處理延遲及音軌裁切偏移等剪輯痛點。

---

## 🌟 核心特色與研究成果

### 1. 高精度暫態鼓點檢測
* 使用 **FFmpeg** 將任何音樂格式秒級轉碼為標準 Mono WAV（低於 0.3 秒）。
* 內建高精度 SciPy/NumPy 頻譜能量暫態鼓點檢測演算法，精準鎖定樂曲重拍與 BPM。

### 2. 裁切偏移量免疫與前奏外推
* 動態讀取時間軸音軌的 `audio_clip.GetLeftOffset()`，自動對齊裁切線，達成 **100% 裁切偏移免疫**。
* 首創**靜音前奏反向外推機制**，在開頭淡入或靜音時依然能以完美節拍開啟第一影格的卡點剪輯。

### 3. 本機 AI 語義選材推薦
* 使用本機、離線的 **CLIP 模型** (`openai/clip-vit-base-patch32`，基於 PyTorch) 提取 65 個影片中 3 張關鍵影格（開頭、中段、結尾）的視覺特徵向量。
* 特徵一次性計算後儲存至本地快取 **`video_metadata.pkl`**（約 2.5MB），之後啟動即享**低於 0.05 秒的極速語義檢索**！
* 使用 OpenCV 計算中段連續 10 幀的平均影格像素變化率（Optical Flow Proxy），精準量化每個影片的物理動態運動強度。

### 4. 嚴格「無重複模式 (Strict Zero-Repetition Mode)」與高強度語義對比
* 🛠️ **終極解決重用率過高與素材過少的問題**：在媒合時，每選定一個影片片段就將其從可用庫中完全 Pop 彈出！這保證了在 30 秒的 61 個卡點上，**全片 61 個片段 100% 完全不同（0 重複、無影格衝突）**，極致豐富地調用了您硬碟中幾乎所有的 65 個獨家相機素材！
* 🛠️ **語義對比度重新歸一化**：對 CLIP 的 Cosine 相似度得分在剩餘庫中進行 `[0.0, 1.0]` 的動態重新歸一化，完美防禦相似度數值過近而被運動強度分數「霸凌」的數學 Bug，確保語義搜尋享有 70% 的絕對掌控權！
* 🛠️ **優化高反差英語 Prompt**：採用專業優化的純英文 Prompt，徹底釋放 CLIP-ViT 在英文資料集上訓練的超凡語義檢索力。

### 5. `math.ceil` 跨影格率無縫對齊
* 解決 **29.97 FPS** 素材在 **24.0 FPS** 時間軸上因浮點數捨入相位差造成的 1 影格黑格問題。
* 採用向上取整動態補償公式：
  $$\text{duration\_source} = \text{int}(\text{math.ceil}(\text{duration\_timeline} \times \frac{\text{src\_fps}}{\text{timeline\_fps}}))$$
  確保單軌 V1 軌道上的影片段落 100% 嚴絲合縫，徹底杜絕畫面閃爍與擠壓。

### 6. 解決 API 座標系天坑
* 修正達芬奇 `AddMarker` 接收相對坐標，而 Timeline 查詢返回絕對座標所導致的 1 小時標記偏離 Bug。
* 完美實現坐標轉換，將標記精準顯示於 `01:00:00:11` 而非 `02:00:00:11`。

### 7. 好萊塢級「視覺領先卡點」
* 自動套用 **`-0.065秒`** 的毫秒級延遲補償，完美抵消 DSP 濾波產生的 69.6ms 群延遲。
* 提供專業剪輯師最愛的 **視覺領先卡點 (Lead-In Cut)**，讓畫面比重音提早 1.5 影格切換，節奏感極度清脆！

### 8. GUI 強制雙重頁面跳轉刷新
* 自動在 `media` 與 `edit` 頁面間跳轉，強制重建達芬奇 GUI 時間軸快取，即刻顯示剪輯結果。
